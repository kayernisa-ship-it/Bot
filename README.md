# Smart Search Bot

Telegram bot with Gemini AI.

## Commands

/start  
/help  

/addsite movie URL  
/addsite song URL  
/addsite apk URL  

/sites  
/removesite movie URL  

## Search URL formats

Works with:

https://example.com/search?q=

https://example.com/search?keyword=

https://example.com/search/{query}

## Environment Variables

BOT_TOKEN  
GEMINI_API_KEY  
ADMIN_ID
